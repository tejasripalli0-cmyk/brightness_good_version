# Brightness Control

A simple brightness control component built using HTML, CSS, and JavaScript.

## Features

* Adjustable brightness slider (0–100%)
* Real-time brightness updates
* Live brightness percentage display
* Interactive preview area

## How It Works

The user can move the slider to increase or decrease brightness. The preview area updates instantly to reflect the selected brightness level.

## Technologies Used

* HTML
* CSS
* JavaScript
